# 3D Print Quote - 3D 打印自动报价技能

自动解析 STL/3MF 模型文件，计算体积、重量、打印时间，生成专业报价。

## 功能

- **STL 解析** - Binary 和 ASCII 格式自动检测
- **3MF 解析** - 支持 ZIP 压缩包结构
- **几何分析** - 体积、表面积、边界盒
- **成本计算** - 材料 + 机器 + 人工 + 利润
- **多材料支持** - PLA/ABS/PETG/尼龙/TPU/树脂
- **批量报价** - 支持多个文件同时分析
- **PDF 报价单** - 生成专业报价文档

## 安装依赖

```bash
pip install numpy  # 可选，加速计算
```

本技能使用纯 Python 实现，无需外部库即可运行。

## 使用方法

### 命令行

```bash
# 分析单个 STL 文件
python ~/.openclaw/workspace/skills/3d-print-quote/main.py model.stl

# 指定材料
python ~/.openclaw/workspace/skills/3d-print-quote/main.py model.stl ABS

# 分析 3MF 文件
python ~/.openclaw/workspace/skills/3d-print-quote/main.py model.3mf

# 批量分析
python ~/.openclaw/workspace/skills/3d-print-quote/main.py *.stl

# 生成 PDF 报价单
python ~/.openclaw/workspace/skills/3d-print-quote/main.py model.stl --pdf quote.pdf
```

### Python API

```python
from skills_3d_print_quote import analyze_model, calculate_quote

# 分析模型
stats = analyze_model('model.stl')

# 计算报价
quote = calculate_quote(stats, material='PLA', profit_margin=1.5)
print(f"建议售价：¥{quote['price']:.2f}")
```

## 支持的材料

| 材料 | 密度 (g/cm³) | 单价 (元/g) | 特点 |
|------|-------------|------------|------|
| PLA | 1.24 | 0.08 | 常用，环保 |
| ABS | 1.04 | 0.10 | 强度高 |
| PETG | 1.27 | 0.12 | 耐候性好 |
| Nylon | 1.14 | 0.25 | 耐磨 |
| TPU | 1.21 | 0.18 | 柔性 |
| Resin | 1.10 | 0.15 | 光固化 |

## 配置

编辑 `config.json` 自定义价格参数：

```json
{
  "machine_cost_per_hour": 5.0,
  "labor_cost_base": 10.0,
  "default_profit_margin": 2.0,
  "default_layer_height": 0.2,
  "default_infill": 20
}
```
